# Roberto Millan Portfolio

Minimal one-page portfolio built with Astro and TypeScript.

## Structure

```text
src/
  components/
    About.astro
    Contact.astro
    Experience.astro
    Footer.astro
    Hero.astro
    Projects.astro
  data/
    experience.ts
    projects.ts
  layouts/
    BaseLayout.astro
  pages/
    index.astro
  styles/
    global.css
```

## Commands

- `npm install`
- `npm run dev`
- `npm run check`
- `npm run build`

## Notes

- The site is static and does not use client-side JavaScript.
- Content is intentionally centralized in the two data files for easy maintenance.
